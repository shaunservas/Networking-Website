Components to a Home Network

Author: Shaun Servas

Now I know what you are thinking, oh my god there is so many components how am I supposed to know what does what. Well this page was designed just for that.

- WAP: WAP stands for wireless access point, this allows wireless devices to connect to a wired network using wi-fi.

- NAS: NAS stands for Network attached storage. This is a server committed to storing computer data. This is connected to your network to store files.

- Modem: A modem is a device that allows you to transmit data from your device. A modem converts digital, which is what your computer uses, to analogue, which is what telephone lines and etc. use.

- A router is a device that pushes data packets through networks. It uses protocols, headers and forwarding tables to find the best route between two networks.

- A switch is a device used to filter and forward packets to devices on a LAN.

- LAN stands for local area network, which connects computers in a relatively small area/ vicinity.

- WAN stands for wide are network, which connects computers worldwide.

- NIC stands for Network interface controller. This connects a computer to a computer network.

- Ethernet is a system that connects computers, forming a LAN.

- 802.11x: 802.llx is a form of wireless LAN.

-IPV4 & IPV6: IPV4 stands for internet protocol version 4, which is how data is sent from one computer to another on the internet. Internet protocol 6 is an upgrade to ipv4.

- DHCP stands for dynamic host configuration protocol. This gives an ip host its ip address and other information such as default gateway, and subnet mask.

- DNS is your domain name system. This holds your domain names website and email settings. DNS allows people to visit your server.

- A gateway is a device used to connect two networks, such as a LAN to the internet.

- A netmask is a line of 1s and 0s which hides the network part of an ip address so that you can only see the host computer part of the address.
